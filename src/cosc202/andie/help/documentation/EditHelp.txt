<p><h1>Edit</h1>
<h2>What Each Button in the Edit Menu does:</h2></p>
<p><b>Undo</b>
Undoes the most recent ImageOperation applied</p>
<p><b>Redo</b>
Redoes the most recent ImageOperation that was undone</p>
<p><b>Resize</b>
Resizes an image according to a scale retrieved from the user
The scale must be between 0 and a displayed limit based on the current size of the image</p>  
<p><b>Flip Horizontal</b>
Flips image using vertical reference line</p>
<p><b>Flip Vertical</b>
Flips image using horizontal reference line</p>
<p><b>Rotate 90° Right</b>
Turns image 90 degrees to the right</p>
<p><b>Rotate 90° Left</b>
Turns image 90 degrees to the left</p>
<p><b>Rotate 180°</b>
Turns the image 180 degrees so it is upside down, but not vertically flipped</p>
