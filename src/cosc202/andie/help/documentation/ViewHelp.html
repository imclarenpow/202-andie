What Each Button Does in the View Menu:

Zoom In: Increases zoom by 10% Shortcut: ctrl + scroll up

Zoom Out: Decreases zoom by 10% Shortcut: ctrl + scroll down

Zoom to Fit Window Size: Zooms in/out of the image so that it fills ANDIE's current window

Zoom Full: Resets the zoom to the image's actual size

Resize Window to Image Size: Resizes ANDIE's window to match the current image size
