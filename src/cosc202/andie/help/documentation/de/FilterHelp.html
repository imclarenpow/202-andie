Filter

Was jeder Button im Filter-Menü macht:

Schärfen Wendet einen Schärfungsfilter auf das Bild an

Medianfilter Wendet einen Medianfilter auf das Bild an

Mittelwertfilter Wendet einen Mittelwertfilter auf das Bild an

Gaußscher Filter Wendet einen Gaußschen Weichzeichnungsfilter auf das Bild an Verwendet einen Schieberegler, um die Stärke der Weichzeichnung anzupassen

Weicher Weichzeichner Wendet einen weichen Weichzeichnungsfilter auf das Bild an
