Editar

Función de cada botón en el menú de edición:

Deshacer Deshace la operación más reciente aplicada a la imagen Atajo: ctrl + z (Command + Shift + Z (Mac))

Rehacer Rehace la operación más reciente que fue deshecha Atajo: ctrl + shift + z (Command + Shift + Z (Mac))

Redimensionar Cambia el tamaño de una imagen según una escala ingresada por el usuario La escala debe estar entre 0 y un límite mostrado basado en el tamaño actual de la imagen

Voltear horizontalmente Voltea la imagen utilizando una línea de referencia vertical

Voltear verticalmente Voltea la imagen utilizando una línea de referencia horizontal

Rotar 90° hacia la derecha Rota la imagen 90 grados hacia la derecha

Rotar 90° hacia la izquierda Rota la imagen 90 grados hacia la izquierda

Rotar 180° Rota la imagen 180 grados para que quede boca abajo, pero no volteada verticalmente
