ファイル

ファイルメニューの各アクションの役割:

開く 画像を開く もし前に編集してANDIEで保存されている画像であれば、編集内容が表示されます ショートカット: ctrl + o (Command + O (Mac))

保存 画像と適用された画像操作を別々のファイルに保存します ショートカット: ctrl + s (Command + S (Mac))

コピーして保存 画像と適用された画像操作を別の場所/名前で別々のファイルに保存します ショートカット: ctrl + shift + s (Command + Shift + S (Mac))

コピーをエクスポート 編集した画像のバージョンを指定した場所に保存します

終了 ANDIEを終了します
