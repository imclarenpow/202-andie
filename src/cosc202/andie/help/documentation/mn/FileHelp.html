Файл

Файл Менюд хийх үйлдлүүдийн үндсэн ажил:

Нээх Зургийг нээнэ Хэрэв зургийг өмнө нь засварласан болон ANDIE дээр хадгалагдсан бол, засварлагдсан үр дүн харагдах болно Товч тодорхойлолт: ctrl + o (Command + O (Mac))

Хадгалах Зургийг хадгалж байгаа болон хэрэглэгдсэн зургийн үйлдлүүдийг нэг тус бус файлуудад хадгална Товч тодорхойлолт: ctrl + s (Command + S (Mac))

Хуулах Зургийг хадгалж байгаа болон хэрэглэгдсэн зургийн үйлдлүүдийг шинэ байршлын/нэртэй файлуудад хадгална Товч тодорхойлолт: ctrl + shift + s (Command + Shift + S (Mac))

Хуулах хуулиар Засварласан зургийн хувилбарыг өгөгдсөн байрлалд хадгална
